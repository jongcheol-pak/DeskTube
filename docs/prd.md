# PRD: DeskTube — 유튜브 동영상 배경화면 앱

> 상태: **승인됨 (2026-07-15)** — 이 문서는 고정. 변경은 사용자 합의 필수.

## 목적

윈도우 사용자가 유튜브 동영상을 바탕화면 배경(아이콘 뒤)으로 재생할 수 있게 하는 WinUI 3 데스크톱 앱.
Microsoft Store 배포를 목표로 하며, 멀티 모니터 환경과 저부하(백그라운드 상시 실행) 동작을 지원한다.

## 플랫폼·기술 제약 (질문 라운드 확정)

- **지원 OS**: Windows 11 전용 (Win10은 2025-10 일반 지원 종료)
- **재생 엔진**: WebView2 + 유튜브 공식 IFrame Player API (ToS 준수, Store 심사 안전). 스트림 추출 방식 금지.
- **임베드 도메인**: `www.youtube.com/embed` 사용 (로그인 쿠키 적용을 위해 `youtube-nocookie.com` 사용 불가)
- **화질**: IFrame API의 `setPlaybackQuality`는 no-op — 화질 강제 불가. 간접 제어(렌더 해상도 스케일)만 제공.
- **배포**: MSIX 패키징 + Microsoft Store, WACK 통과 필수
